# Adds a teacher magnitude modulator to RLVR, using privileged self-distillation signals to improve training stability.

## Relevance to This Project
They add an explicit teacher signal to RLVR. We show RLVR already IS self-distillation -- the teacher is implicit in GRPO's group normalization, making the explicit modulator a redundant but empirically useful amplifier.
