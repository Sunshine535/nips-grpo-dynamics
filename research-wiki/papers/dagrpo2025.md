# Filters sequences at the group level to rectify gradient conflicts in GRPO, using distinctiveness scores to remove conflicting gradient signals.

## Relevance to This Project
CSD explains gradient conflicts as distillation/anti-distillation interference: within a group, correct samples distill while incorrect samples anti-distill, and low-distinctiveness groups have nearly canceling signals.
